Linear Regression API

What is this?

This is an API that will perform a linear regression on data which you can import below.

How do I use it?

You can upload data in .csv or .tsv format. Your data should only have two rows, one for the dependent variable (on the top row) and another for your independent variable (on the bottom row). Any other rows will be cut off. See below for an example.

age,43,21,25,42,57,59
glucose,99,65,79,75,87,81

Uploading the above data will tell you the correlation between age and glucose levels absent any other factors.
